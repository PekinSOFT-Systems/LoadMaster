Provides a method of logging warning messages to the log file. 

For this message to be logged, the {\ttfamily \mbox{\hyperlink{classcom_1_1pekinsoft_1_1loadmaster_1_1sys_1_1Logger}{Logger}}} must have been properly established (N\+OT in the {\ttfamily O\+FF} state) and the level must be set at {\ttfamily W\+A\+RN} or {\ttfamily C\+R\+I\+T\+I\+C\+AL}. 

A good practice for this type of message is to place user, system and java information into the message, as well as the information that you were going to display. This will provide a solid base of knowledge for tracking down the reason for the warning.

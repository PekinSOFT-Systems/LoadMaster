This method does not zero the progress bar out once completed, nor do we change its visibility within this method. Therefore, prior to providing the progress bar, the developer needs to make sure that it is visible to the user and, once this method returns, the developer needs to handle the progress bar how they desire.

Furthermore, this method changes the tooltip text of the progress bar to \char`\"{}\+Broker Search Progress\char`\"{}, so the developer will also need to reset the tooltip on the progress bar once this method returns.

◆ postTransactions()

abstract boolean genericstest.AbstractJournal< T >.postTransactions ( ) throws DataStoreException
abstract

This method performs the task of posting all unposted transactions from this journal to the General Ledger.

Developer's Note
This method should only be called when the user clicks the "Post" task on the Accounting task pane, if they have the batch method selected in the application settings dialog's Accounting tab. If they do not have the batch method selected, this method should be called immediately upon any journal entries being made.
Returns
success: true if transactions posted, false otherwise.
Exceptions
DataStoreExceptionin the event there is an error accessing the database.

Detailed Description

The Processor class provides static methods for processing Journals and the General Ledger for Load Master™.

By concentrating all of the account processing methodology in this one class, code maintenance will be made simpler for PekinSOFT™ Systems in the future.

The following Account Journals are processed via exposed methods in this class:

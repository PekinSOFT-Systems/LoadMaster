Sets the unique identification number for this journal model.

Developer's Note
The BaseTransaction class generates a unique identification number for each new instance. Therefore, this setter is only here to allows programs to read in existing BaseTransaction data from a data file on disk. Having said that, of course the default unique ID may be overriden, but caution needs to be taken in doing so, as the built-in unique ID generator guarantees there will never be a repeated ID number.
